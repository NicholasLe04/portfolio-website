  • SCE TV

    Summer 2023

    Used by 300+ SCE members, SCE TV allows members to remotely play YouTube videos on a TV. It uses a Raspberry Pi to receive an RTMP broadcast via VLC Media Player and a FastAPI server that manages the video stream. Users communicate with the backend via a simple HTML page and HTTP requests. The backend is deployed on a server using Docker and monitored with Prometheus.
